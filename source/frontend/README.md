# Kelimelik

Multi-mode Turkish word game app with 5 game modes, social tournaments, TDK word definitions, and AdMob monetization.

## Getting Started

```bash
flutter pub get
flutter run
```

## Requirements

- Flutter 3.41.6+
- Dart 3.9+
- Android minSdk 24

## Package

`com.kelimelik.app`
